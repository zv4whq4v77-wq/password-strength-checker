# Password Strength Checker

This project is a Python program that evaluates the strength of a password based on length and character complexity.

## Features
- Checks if the password meets minimum length requirements
- Detects uppercase letters, numbers, and special characters
- Classifies passwords as Weak, Medium, or Strong

## Technologies Used
- Python

## Purpose
This project was created to apply basic cybersecurity principles and understand how password strength is evaluated in real-world systems.
